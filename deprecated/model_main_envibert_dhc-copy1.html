Model Controller Tutorial: EnviBert model with Deep Hierarchical Classification

This notebook contains some example of how to use the EnviBert-based models in this NLP library

We will walk through other cases of classification: multi-head and multi-label. Since we will showcase the capabiilty of this label in these cases, there won’t be as detailed as this tutorial
